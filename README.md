# GDMirage Server

[![CI](https://github.com/MirageGD/Server/actions/workflows/ci.yaml/badge.svg)](https://github.com/MirageGD/Server/actions/workflows/ci.yaml)
![CodeFactor Grade](https://img.shields.io/codefactor/grade/github/MirageGD/Server)
![GitHub License](https://img.shields.io/github/license/MirageGD/Server)

A lightweight 2D ORPG engine built on Godot and C#.

## Game Client

For the game client see <https://github.com/MirageGD/Client>.

## Game Data

To ensure the server runs correctly, you must place the required game data files
inside the server’s content folder.

For a reference implementation, check out the sample starting world available
here: <https://github.com/MirageGD/Sample-World>
